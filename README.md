# Arduino Fire Fighting Robot

This project is a fire-fighting robot built with Arduino. The robot is designed to detect fire and put it out using a servo mechanism and a water pump.

## Components Used
- Arduino Uno
- Servo Motor
- IR Sensors
- Motors and Motor Driver
- Water Pump
